# Reading Tracker Web Application
## Tech Stack

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite

## Overview

Reading Tracker is a full-stack web application developed as my A-Level Computer Science coursework project. The system enables students to record and monitor their reading progress while providing administrators with tools to manage users, books, and reading records through a dedicated dashboard.

The project demonstrates object-oriented programming, web development, authentication, data management, and software design principles using Python and Flask.

---

## Features

### Student Features

* Secure user authentication and login
* Personal reading dashboard
* Track books read and reading progress
* View individual reading statistics
* Leaderboard comparing reading activity with other students

### Administrator Features

* Secure administrator login
* Manage student accounts
* Add, edit and remove books
* View and manage reading records
* Monitor overall system activity

## Installation

Clone the repository:

```bash
git clone https://github.com/JoshuaLadepo/CSNEA.git
```

Navigate into the project folder:

```bash
cd reading-tracker
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
